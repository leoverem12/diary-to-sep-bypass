# ЄШ Щоденник — швидкий доступ без труднощів 

Цей додаток створено для прямого входу до щоденника через diary.eschool-ua.com, уникаючи автоматичного перенаправлення на sep.eschool-ua.com. 

## Основні можливості: 

- **Видалення автоматичних перенаправлень:** технологія `declarativeNetRequest` блокує перенаправлення зі сторінки diary.eschool-ua.com на sep.eschool-ua.com. 
- **Налаштування cookies:** додаються cookies з параметром `ignoreSep=true`, а також вносяться зміни у `localStorage` або `sessionStorage`, щоб уникнути небажаних переходів. 
- **Управління запитами:** змінюються функції `fetch` та `XMLHttpRequest` з метою видалення параметра `url_redirect` у JSON-відповідях. 
- **Блокування змін адреси:** функції на кшталт `location.assign`, `replace` або зміна поля `href` блокуються, якщо вони ведуть до sep.eschool-ua.com. 

## Як встановити: 

1. Завантажте архів із додатком та розпакуйте його у зручне для вас місце. 
2. Перейдіть у браузері Chrome на сторінку `chrome://extensions/`. 
3. Увімкніть режим розробника за допомогою перемикача у верхньому правому куті. 
4. Натисніть "Завантажити розпакований додаток" і вкажіть шлях до папки із завантаженим додатком. 
5. Після встановлення іконка додатка з’явиться на панелі інструментів браузера, забезпечуючи миттєвий доступ до щоденника. 

Плагін працює для сайту diary.eschool-ua.com, спрощуючи доступ до щоденника та усуваючи зайві кроки.
