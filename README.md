# ЄШ Щоденник — без редиректу

Прибирає автоматичний редирект з diary.eschool-ua.com на sep.eschool-ua.com. Відкривайте щоденник одразу, без зайвих кліків.

## Принцип роботи

- **Блокування редиректу**: Використовує `declarativeNetRequest` для блокування переходів на sep.eschool-ua.com з diary.eschool-ua.com.
- **Куки та сховища**: Встановлює куки `ignoreSep=true` та значення в localStorage/sessionStorage для ігнорування редиректу.
- **Перехоплення запитів**: Перевизначає `fetch` та `XMLHttpRequest`, видаляючи поле `url_redirect` з JSON-відповідей.
- **Перевизначення location**: Блокує `location.assign/replace` та `href` setter для URL з sep.eschool-ua.com.

## Інструкція з встановлення

1. Завантажте всі файли розширення в одну папку.
2. Відкрийте Chrome та перейдіть до `chrome://extensions/`.
3. Увімкніть **Режим розробника** (у правому верхньому куті).
4. Натисніть **Завантажити розпаковане розширення** та оберіть папку з файлами.
5. Готово! Іконка з'явиться у панелі розширень. Клікніть для відкриття щоденника.

Розширення активне автоматично на diary.eschool-ua.com.
