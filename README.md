# Обхід SEP для щоденника Єдиної Школи

Цей додаток створено для прямого входу до щоденника через diary.eschool-ua.com, уникаючи автоматичного перенаправлення на sep.eschool-ua.com (єдину точку входу). 
Ви можливо замічали, що потрібно швидко зайти на щоденник, а у висновку кожен раз авторизовуватися в єдину точку входу, а потім вже ще раз входити але в щоденнику.
Данний плагін приберає перенаправлення на SEP (ЄТВ), і без танців з бубном сразу авторизуватися в щоденнику.

## Як встановити: 

1. Завантажте архів із додатком та **розпакуйте його** у зручне для вас місце. 
2. Перейдіть у chromium-браузері (Chrome, Brave, Edge) на сторінку `chrome://extensions/`. (автоматично підставиться відповідно до вашого браузера)
3. Увімкніть режим розробника за допомогою перемикача у верхньому правому куті. 
4. Натисніть "Завантажити розпакований додаток" і вкажіть шлях до папки із **розпакованим плагіном**. 
5. Після встановлення іконка додатка з’явиться на панелі інструментів браузера, забезпечуючи миттєвий доступ до щоденника.
6. Все готово.


## Видалення:
Якщо ви хочете **видалити** плагін, то
1. Перейдіть на вже знайому сторінку `chrome://extensions/`, і натисніть біля плагіна кнопку "Видалити".
2. Підвердіть дію.
3. **Перезапустіть браузер!**
4. Все готово.


Я не додам цей плагін у магазин Chrome так як там просять ЦІЛИХ 5 доларів для створення акаунту.
Але **не видаляйте папку з плагіном**, бо тоді плагін перестане працювати.
